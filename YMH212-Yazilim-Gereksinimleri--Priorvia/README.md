# YMH212-Yazilim-Gereksinimleri--Priorvia

Priorvia is a simple, fast, and powerful task management platform designed specifically for student teams and small groups. Say goodbye to complex tools and hello to streamlined productivity.

Technologies Used

HTML5: Semantic structure and accessible elements.

CSS3: Custom variables, Flexbox layouts, animations, and responsive design.

JavaScript: DOM manipulation, event listeners, and modal state management
